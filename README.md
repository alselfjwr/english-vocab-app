# English — 英語 基本単語1000

日本語話者向けの英語学習アプリ。基本単語1000語を「単語帳・フラッシュカード・クイズ」の3つのモードで学習できます。

[vietnamese-vocab-app](https://vietnamese-vocab-app.vercel.app/) の英語学習バージョンです。

## 機能

- 📖 **単語帳** — 37カテゴリ・1000語の一覧、検索、お気に入り（★）登録
- 🃏 **フラッシュカード** — 英語→日本語／日本語→英語の両方向、シャッフル、自動発音
- ✏️ **クイズ** — カテゴリ別の4択クイズ（10問）、間違えた単語の復習表示
- 💬 **会話** — 日常会話13シーン・156フレーズ＋シーン別の対話練習（ロールプレイ）
  - 「見る・聞く／Aを演じる／Bを演じる」を選んで会話をステップ再生
  - 自分の番は日本語ヒントだけ表示 → タップで英文を確認
  - 🎤 **発音チャレンジ** — 音声認識（SpeechRecognition）でフレーズの発音を判定
  - 会話フレーズはフラッシュカード・クイズのカテゴリとしても選択可能
- 🔊 **発音再生** — Web Speech API による英語（en-US）読み上げ、標準／ゆっくり切り替え

## 技術

- 依存なしの単一HTMLファイル（`index.html`）
- ビルド不要・静的ホスティングのみで動作
- お気に入りは localStorage に保存

## 開発

```sh
npx serve .
```
